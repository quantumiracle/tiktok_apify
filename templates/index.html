About TikTok Parser

The parser extracts the following data for each profile:

  • Username
  • Profile URL
  • Follower Count
  • Total Likes
  • Following Count
  • Friends Count
  • Video Count
  • Bio Text
  • Email Address (if found)

Note: Running the parser may take some time depending on the number of topics and profiles.
